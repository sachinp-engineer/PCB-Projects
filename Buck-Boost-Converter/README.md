# ⚡ Buck-Boost Converter PCB Design

A PCB implementation of a DC-DC Buck-Boost Converter designed using KiCad.

This converter can either increase (boost) or decrease (buck) the input voltage depending on the operating conditions.

---

## 📌 Project Overview

This project includes:

- Dual PWM Signal Generator using LM555 Timer ICs
- Buck-Boost Converter Stage
- MOSFET Switching Circuit
- Compact PCB Layout
- 3D PCB Design

## ⚙ Working Principle

- Generates PWM signals using LM555 timer circuits.
- Controls IRF540N MOSFET switching.
- Operates in Buck mode when output voltage is lower than input.
- Operates in Boost mode when output voltage is higher than input.
- Smooth output voltage using inductors and capacitors.
