# 📌 Projects

## ⚡ 1. Boost Converter PCB

A DC-DC Boost Converter designed using KiCad that steps up a lower DC input voltage to a higher output voltage.

### Features

- LM555 PWM Generator
- IRF540N MOSFET Driver
- Compact PCB Layout
- Single-Layer PCB
- 3D PCB Model

📁 Folder:
```
Boost-Converter/
```

---

## ⚡ 2. Buck-Boost Converter PCB

A DC-DC Buck-Boost Converter capable of both stepping up and stepping down the input voltage using PWM control.

### Features

- Dual LM555 PWM Circuits
- IRF540N MOSFET
- Multiple Inductor Stages
- Compact PCB Design
- 3D PCB Visualization

📁 Folder:
```
Buck-Boost-Converter/
```
