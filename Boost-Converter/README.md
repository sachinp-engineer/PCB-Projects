# ⚡ Boost Converter PCB Design

This project demonstrates the design of a **Boost Converter PCB** using **KiCad**. The converter steps up a lower DC input voltage to a higher DC output voltage using a PWM-controlled switching circuit.

---

## 📌 Project Overview

The PCB consists of two main sections:

- PWM Signal Generator using LM555 Timer
- DC-DC Boost Converter Stage

The PWM signal drives the MOSFET switch, allowing energy storage in the inductor and increasing the output voltage.

## ⚙ Working Principle

1. LM555 generates a PWM signal.
2. PWM drives the IRF540N MOSFET.
3. The MOSFET switches the inductor ON and OFF.
4. Energy stored in the inductor is transferred through the diode.
5. The capacitor smooths the boosted output voltage.
